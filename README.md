# Real-time-chat

## Docker
> docker-compose up --build

### 启动
> docker start real-time-chat-frontend
> docker start real-time-chat-backend

### 关闭
> docker stop real-time-chat-backend
> docker stop real-time-chat-frontend

## backend
- go
> go run main.go
## frontend
- Nextjs
> bun install
> bun run dev
